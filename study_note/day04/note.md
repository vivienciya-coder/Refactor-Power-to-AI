### 3.2 坑位与 Debug (Q&A)

**问题 1：LangChain 模板变量在大括号前紧贴英文冒号导致报错**

*   **描述**：在使用 `LangChain` 的 `PromptTemplate.from_template` 定义提示词模板时，输入 :(英文版)会报错 输入 ：（中文版）不会报错
    1.  **语法冲突**：在 Python 的字符串格式化标准中，`{variable:format}` 具有特殊含义（如 `{price:.2f}` 用于限制小数位）。
    2.  **解析歧义**：LangChain 模板引擎 会误认为在进行“变量格式化操作”而非简单的变量替换。因冒号后没有合法的格式化指令，导致引擎崩溃。
*   **解决**：
    2.  **符号替代**：改用中文冒号。✅ 示例：`原因 ：`
